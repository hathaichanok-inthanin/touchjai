# TouchJai - Automotive Customer Loyalty & Membership System
ระบบสมาชิกและสะสมแต้ม สำหรับศูนย์บริการรถยนต์ครบวงจร

## Project Overview
TouchJai เป็นระบบบริหารจัดการความสัมพันธ์ลูกค้า (CRM) ที่ออกแบบมาเพื่อเปลี่ยนลูกค้าขาจรให้เป็นลูกค้าประจำ (Loyalty Program) โดยระบบเน้นการจัดเก็บประวัติการใช้บริการ การสะสมคะแนนจากยอดซื้อ และการมอบสิทธิประโยชน์เฉพาะบุคคล เพื่อสร้างความพึงพอใจสูงสุดให้กับผู้เข้ารับบริการในเครือเอกการยาง

## Tech Stack
* **Framework:** Laravel 5.7 (PHP)
* **Database:** MySQL
* **Frontend:** Bootstrap, jQueryฃ
* **Features:** * Member Registration & Profile (ระบบสมัครและจัดการข้อมูลสมาชิก)
    * Point Accumulation & Redemption (ระบบสะสมและแลกคะแนน)ฃ
    * Tiered Membership (ระบบระดับสมาชิกตามยอดใช้จ่าย)

## Key Features & Logic Highlights

* **Centralized Member Database:** ระบบฐานข้อมูลสมาชิกที่เชื่อมโยงทุกสาขาเข้าด้วยกัน ทำให้ลูกค้าสามารถใช้สิทธิ์หรือสะสมแต้มได้ไม่ว่าจะเข้าใช้บริการที่สาขาใด
* **Smart Point Calculation:** ระบบคำนวณคะแนนอัตโนมัติจากยอดชำระเงินจริง 
* **Detailed Service Timeline:** บันทึกประวัติการเปลี่ยนยาง (รุ่นยาง, วันที่เปลี่ยน, เลขไมล์)
